### **A Generative AI-driven battle arena where coding syntax is your weapon. Built with Gemini 1.5 Flash to gamify multi-language technical education.**


**Code Colosseum** is a high-octane, gamified educational platform that turns dry coding exercises into an immersive boss battle. Using **Google Gemini 1.5 Flash**, the game procedurally generates unique technical challenges, forcing players to use their coding knowledge as a weapon to survive.

Traditional coding platforms often feel static and intimidating. **Code Colosseum** bridges the gap between learning and engagement by providing :

* **Dynamic Storytelling:** No two battles are the same. Every monster, task, and technical constraint is generated on-the-fly by AI.
* **Instant Feedback:** A specialized terminal provides immediate compiler-style feedback, turning errors into "dodgeable" boss attacks.
* **Polyglot Mastery:** Integrated support for **Java and Python**, allowing users to compare logic across different syntax structures.
